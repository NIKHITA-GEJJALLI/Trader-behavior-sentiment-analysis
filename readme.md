Trader Behavior vs Market Sentiment

This project analyzes the relationship between Bitcoin market sentiment
(Fear & Greed Index) and trader behavior using historical trade data
from Hyperliquid.

Objective
To understand how market sentiment influences trader profitability
and win rates, and to extract insights that can support sentiment-aware
trading strategies.

Datasets
- Hyperliquid Historical Trader Data
- Bitcoin Fear & Greed Index

Key Findings
- Average trader profitability increases as sentiment shifts from Fear to Greed
- Extreme Greed periods show the highest win rates
- Extreme Fear markets exhibit lower profitability and higher uncertainty

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
